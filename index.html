    <div id="beef-curry" class="recipe-page active">

        <img src="https://source.unsplash.com/800x600/?gourmet,beef,curry,dark" alt="Smoked Beef Curry" class="recipe-illustration">


        <div class="recipe-header">

            <h2>煙燻巴沙米克牛肋咖哩</h2>

            <div class="recipe-meta">RICH • SMOKY • SLOW BRAISED</div>

        </div>


        <div class="section">

            <div class="section-title">食 材</div>

            <ul class="ingredients-grid">

                <li><span>優選牛肋條 (修筋剪小塊)</span> <span>450g-600g</span></li>

                <li><span>焦糖洋蔥 (0.3cm 極細丁)</span> <span>1 顆</span></li>

                <li><span>原野蘑菇 (厚切片)</span> <span>8 朵</span></li>

                <li><span>手磨蒜泥 / 生薑泥</span> <span>1 大匙 / 1 小匙</span></li>

                <li><span>煙燻紅椒粉 / 秘製咖哩粉</span> <span>各 1 大匙</span></li>

                <li><span>熟成巴沙米克醋 (Balsamic)</span> <span>1.5 大匙</span></li>

                <li><span>澄清雞高湯</span> <span>適量 (蓋過食材)</span></li>

                <li><span>70% 黑巧克力 / 爪哇咖哩塊</span> <span>2 片 / 1/4 塊</span></li>

            </ul>

        </div>


        <div class="section">

            <div class="section-title">烹飪</div>

            <ul class="technique-list">

                <li data-step="I">

                    <strong>美拉德反應 (Maillard Reaction)</strong>

                    高火熱鍋，不加油，將牛肋條塊每一面煎至極致金黃焦脆。盛出備用，保留鍋內珍貴牛油。

                </li>

                <li data-step="II">

                    <strong>蔬菜與菌菇的底蘊</strong>

                    利用原鍋牛油，低火慢炒洋蔥丁直至微透明焦糖化。加入蘑菇煸炒至出水縮小，香氣濃縮。

                </li>

                <li data-step="III">

                    <strong>辛香料的綻放</strong>

                    補入少許橄欖油，爆香蒜泥與薑泥。撒入紅椒粉與咖哩粉，翻炒 30 秒使油脂充分吸收粉末香氣。

                </li>

                <li data-step="IV">

                    <strong>巴沙米克醋洗鍋 (Deglazing)</strong>

                    倒入牛肉與醋液。利用酸性刮起鍋底凝結的肉香物質（Fond），這是醬汁深度的靈魂來源。

                </li>

                <li data-step="V">

                    <strong>長效慢火燉煮 (L'art de Mijoter)</strong>

                    注入雞高湯。大火煮沸後轉極小火，蓋緊鍋蓋悶煮 60 分鐘。

                    <em>* 於 30 分鐘處檢查水位，若低於食材需酌量補充高湯。</em>

                </li>

                <li data-step="VI">

                    <strong>最終熟成與巧克力</strong>

                    熄火，利用餘溫溶入爪哇咖哩塊。重新開火悶煮 10 分鐘進行濃縮收汁。最後拌入黑巧克力，賦予醬汁深邃色澤與醇厚苦甜感。

                </li>

            </ul>

        </div>


        <div class="chef-note">

            此料理精髓在於「酸度的轉化」。長達一小時的悶煮能將巴沙米克醋的銳利轉為果乾般的甜美。若試味仍覺酸度明顯，可滴入極少許蜂蜜平衡。

        </div>

        <div class="recipe-header">

            <h2>韓式豆腐義大利麵</h2>

            <div class="recipe-meta">FUSION • ARTISAN • TEXTURED</div>

        </div>


        <div class="section">

            <div class="section-title">食 材</div>

            <ul class="ingredients-grid">

                <li><span>板豆腐 (需預先壓重去水 30m)</span> <span>1 塊</span></li>

                <li><span>義大利麵 (細圓麵為佳)</span> <span>25g (精緻擺盤)</span></li>

                <li><span>韓式風味醬 (辣醬/醬油/蠔油/糖)</span> <span>各 1 大匙</span></li>

                <li><span>洋蔥碎 / 蒜泥 (極細化處理)</span> <span>適量</span></li>

                <li><span>芝麻香油 / 冷壓橄欖油</span> <span>數滴 / 1 大匙</span></li>

            </ul>

        </div>


        <div class="section">

            <div class="section-title">烹飪</div>

            <ul class="technique-list">

                <li data-step="I">

                    <strong>豆腐盅結構工程</strong>

                    板豆腐壓乾水分後，以圓形模具挖空中心，取薄片填補底部。此步驟為承載麵條的關鍵地基。

                </li>

                <li data-step="II">

                    <strong>上火炙烤 (Glazing)</strong>

                    將醬料乳化後均勻塗抹於豆腐內外。置於上火烤箱 180°C 烘烤 10-12 分鐘，直至表面呈現琥珀焦糖感。

                </li>

                <li data-step="III">

                    <strong>義大利麵與醬泥</strong>

                    滾水加入 1% 鹽度。麵條提前 2 分鐘出鍋（Al Dente）。鍋中炒香極細洋蔥與醬汁，加入保留的 50ml 麵水乳化。

                </li>

                <li data-step="IV">

                    <strong>立體擺盤與點綴</strong>

                    使用長叉將麵條捲成高聳狀填入豆腐。淋上幾滴芝麻油，撒上蔥綠絲或辣椒絲增加視覺張力。

                </li>

            </ul>

                <li><span>和風醬油 / 清酒 / 味霖</span> <span>3 / 2 / 0.5 大匙</span></li>

                <li><span>軟水 (適宜長時間燉煮)</span> <span>600ml</span></li>

        <div class="section">

            <div class="section-title">La Technique / 烹飪工藝</div>

            <ul class="technique-list">

                <li data-step="I">

                    <strong>紅蘿蔔的預處理</strong>

                    紅蘿蔔先行入滾水煮 10 分鐘，軟化堅硬纖維，確保與馬鈴薯同步熟成。

                </li>

                <li data-step="II">

                    <strong>肉質嫩度控管</strong>

                    大火炒香洋蔥，加入牛五花片炒至 7 分熟。隨即撈起肉片，避免長時間燉煮導致蛋白質過度收縮而乾柴。

                </li>

                <li data-step="III">

                    <strong>慢火 braise 與滲透</strong>

                    加入馬鈴薯塊翻炒，倒入液體調料。中火煮沸後蓋蓋悶煮 20 分鐘。

                </li>

                <li data-step="IV">

                    <strong>最終熟成 (The Resting)</strong>

                    確認馬鈴薯鬆軟後回填肉片。**關火悶 10 分鐘**，利用餘溫讓肉片吸飽馬鈴薯澱粉釋出的濃醇湯汁。

                </li>

            </ul>
